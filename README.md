# AI Dev Playbook

Practical notes, workflows, and repeatable patterns for AI-assisted software development.

This repository is for things that are easy to forget:

- Local workspace conventions
- Git hygiene for AI tooling
- Prompting and collaboration patterns
- Delivery checklists
- Reusable templates

## Why This Repo Exists

Useful fixes and working patterns often disappear into local chat history, temporary notes, or one-off project context. This repo is the long-term place to keep those lessons searchable, reusable, and shareable.

## Structure

- `docs/`: focused write-ups for one practice or workflow
- `templates/`: reusable note templates for future posts

## Recommended Workflow

1. When a problem is solved, turn the outcome into a short note.
2. Save one concrete rule, one example, and one boundary.
3. Publish the note here instead of leaving it in local chat history.
